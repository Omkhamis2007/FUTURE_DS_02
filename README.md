# 🚀 FUTURE_DS_02 - Customer Churn Analysis

## 📌 Project Overview
Customer churn is one of the most critical challenges in subscription-based businesses. This project analyzes customer data to identify churn patterns, understand customer behavior, and provide actionable strategies to improve retention.

An interactive dashboard has been developed using Power BI to present insights in a clear and business-focused manner.

---

## 🎯 Objectives
- Analyze customer churn trends  
- Identify high-risk customer segments  
- Understand retention patterns  
- Provide data-driven business recommendations  

---

## 🛠️ Tools & Technologies
- Power BI (Dashboard & Visualization)  
- Data Analysis Techniques  
- Data Cleaning & Transformation  

---

## 📊 Dashboard Highlights
- 📌 KPI Cards: Total Customers, Churn Customers, Churn Rate  
- 📊 Bar Chart: Churn by Contract Type  
- 🥧 Pie Chart: Customer Distribution (Churn vs Retained)  
- 🎛️ Interactive visuals for better analysis  

---

## 🔍 Key Insights
- Month-to-month customers show the highest churn behavior  
- New customers (low tenure) are more likely to leave early  
- Long-term contract customers demonstrate strong retention  

---

## 💡 Business Recommendations
- Encourage long-term subscriptions with attractive offers  
- Improve onboarding experience for new users  
- Implement loyalty and retention programs  
- Analyze pricing strategy for high-churn segments
